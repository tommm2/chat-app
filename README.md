# Vue 3 + Firebase 即時聊天室

簡介：
* 練習 Vue 3 與 Firebase 製作的即時聊天室

使用技術：
* Firebase Firestore、Firebase Auth
* SCSS
* Vite
* Vee Validate 4 驗證表單
* Vue 3 Composition API

功能：
* 登入、登出功能
* 註冊帳號功能
* 第三方登入(Facebook、Google)
* 進入聊天室聊天
